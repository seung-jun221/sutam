초중등 과정부터 고등 연계까지, 체계적인 7단계 교육 시스템

초등 EA → ES → EF
중등 MS → MD
고등 TI → TE

초등 과정

EA반 선택

초등교과 입문 6개월 과정

+ 영재성 토론수업

학습내용: 초4-1, 초4-2 (응용, 최상위S) 총 4권 + 영재성 토론수업

입학대상: 예비 초4 자기학년 진도 학생

ES반

초등교과 집중 6개월 과정

학습내용: 초5-1 ~ 초6-2 (기응, 최상위S) 총 8권

입학대상: 초등 4학년 이상 초등과정 진행 중인 학생

EF반 선택

초등경시 심화 6개월 과정

+ 실전 경시 기출

학습내용: 초5 경시 2개월 / 초6 경시 2개월 / 실전 경시 2개월

입학대상: 초등 4학년 이상 초등과정 학습 완료 학생

중등 과정

MS반

중등교과 입문 6개월 과정

+ 프리모노 과정

학습내용: 중1-1 기본 2개월 / 심화 2개월 / 중1-2 기본 2개월

입학대상: 초등 4학년 이상 초등과정 학습 완료 학생

MD반

모노-다이 8개월 과정

+ 중등 교과 진도

학습내용: 중2-1, 중2-2, 중3-1, 중3-2 교과 진도 병행

입학대상: 중1 과정 심화 학습 완료 학생

고등 과정

TI반

트라이 4개월 과정

+ 고등 교과 진도

학습내용: 공통수학1 교과 진도 병행

입학대상: 중등 심화 학습 완료 학생

TE반

테트라 6개월 과정

+ 고등 교과 진도 + 파이널 실전 모의고사

학습내용: 공통수학2 교과 진도 병행

입학대상: 공통수학1 2회독 이상 완료 학생
